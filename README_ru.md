Поставьте в очередь сотни промптов, генерируйте видео **VEO 3** и изображения **Nano Banana** массово, и автоматически загружайте каждый результат — больше не нужно сидеть над каждым промптом по отдельности.

**Ключевые слова этого руководства:** автоматизация Google Flow · автоматизация VEO · пакетная генерация VEO 3 · пакет изображений Nano Banana · массовый генератор ИИ-видео · автоматическая загрузка видео VEO · расширение Chrome для Google Flow.

## 🚀 Зачем нужен VEO Automation

Google Flow — мощный инструмент, но официальный интерфейс заставляет вас запускать **один промпт за раз** и скачивать **по одному файлу за раз**. Если вы создаёте контент в больших объёмах — короткие ролики, рекламные креативы, раскадровки, продуктовые кадры — этот ручной цикл становится узким местом.

VEO Automation убирает его:

- **Вставьте список и занимайтесь своими делами.** Добавьте десятки или сотни промптов — расширение обработает их последовательно, с повторными попытками, пока вы заняты чем-то другим.
- **Видео VEO 3 *и* изображения Nano Banana в одном инструменте.** Большинство автоматизаций работают только с видео. Это расширение пакетно обрабатывает и видеомодели Google Flow, и модель изображений Nano Banana — из одной боковой панели.
- **Файлы сохраняются на диск автоматически.** Каждое готовое видео и изображение переименовывается и сохраняется в отдельную папку проекта — никаких марафонов с «сохранить как».

Если вы искали *пакетный инструмент для Google Flow*, *массовый генератор VEO 3* или способ *автоматически загружать изображения Nano Banana* — это создано именно для вас.

## 📥 Установка

### Chrome Web Store (рекомендуется)

1. Откройте [**страницу VEO Automation**](https://chromewebstore.google.com/detail/veo-flow-automation-%E2%80%94-bat/inmkimakahiennggieafocbmfjlmfbhn?hl=ru) в Chrome Web Store.
2. Нажмите **Установить** → **Установить расширение**.
3. Закрепите расширение: нажмите на иконку-пазл на панели инструментов Chrome, затем на значок закрепления рядом с **VEO Automation**, чтобы в следующий раз оно было в один клик.

> **Совместимость:** Chrome 137+ (и браузеры на базе Chromium, такие как Edge и Brave). Расширение открывается в виде **боковой панели** рядом с Google Flow.

## ⚡ Быстрый старт

1. **Откройте Google Flow.** Перейдите на [labs.google/fx/tools/flow](https://labs.google/fx/tools/flow) и откройте (или создайте) проект. Расширение активируется только на страницах проектов Flow.
2. **Откройте боковую панель.** Нажмите на иконку VEO Automation. Войдите в аккаунт, чтобы разблокировать дневную квоту.
3. **Выберите режим.** Выберите один из пяти режимов генерации (см. ниже).
4. **Добавьте промпты.** Введите их или загрузите файл `.txt`.
5. **Нажмите «Запустить».** Наблюдайте, как очередь обрабатывает каждую задачу, повторяет попытки при ошибках и автоматически загружает результаты.

> ℹ️ **Про баннер «отладки».** Пока расширение работает, Chrome показывает сверху полосу *«… отлаживается»*. Это необходимо: Google Flow принимает только настоящий, доверенный пользовательский ввод, поэтому расширение управляет страницей через официальный интерфейс отладки Chrome. Это не вредоносная программа — это цена надёжной автоматизации. Не закрывайте баннер, пока выполняется пакет.

## 🎛️ Режимы генерации

### 1. Текст в видео — VEO 3

Массово создавайте видео прямо из текста.

1. Выберите **Текст в видео**.
2. Введите промпты в поле (разделяйте каждый промпт **пустой строкой**) **или** нажмите **Загрузить .txt**, чтобы импортировать список промптов.
3. Задайте длительность (один клип 8 с или **Concat** для более длинных сцепленных видео).
4. Нажмите **Запустить**.

**Пример списка промптов:**

```
Одинокий маяк на утёсе в час золотого света, волны разбиваются внизу, медленный кинематографичный наезд камеры.

Освещённый неоном переулок Токио под дождём, отражения на мокром асфальте, поднимается пар, ощущение съёмки с рук.

Аэросъёмка над осенним лесом, низкий утренний туман между деревьями, тёплый свет.
```

### 2. Изображение в видео — VEO 3

Оживите статичные кадры.

1. Выберите **Изображение в видео**.
2. Нажмите **Выбрать изображения** (поддерживается множественный выбор) и перетаскивайте их, чтобы упорядочить по имени, дате или вручную.
3. Напишите промпт для каждого изображения (разделяя пустой строкой).
4. Выберите длительность, затем нажмите **Запустить**.

> **Совет:** используйте начальный + конечный кадр, чтобы управлять тем, как начинается и заканчивается движение. Настройте 1–2 изображения на промпт в Настройках.

### 3. Текст в изображение — Nano Banana

Генерируйте изображения по описаниям, пакетно.

1. Выберите **Текст в изображение**.
2. Введите детальные промпты для изображений (разделяя пустой строкой).
3. Задайте соотношение сторон и качество изображения в Настройках.
4. **Запустите** — каждое изображение автоматически загружается в выбранном разрешении (1K / 2K / 4K).

### 4. Изображение в изображение — Nano Banana

Редактируйте или меняйте стиль референсных изображений в любых масштабах.

1. Выберите **Изображение в изображение**.
2. Загрузите исходные изображения.
3. Напишите промпты преобразования (например, *«тот же персонаж, киберпанк-наряд, крыша ночью»*).
4. **Запустите**, чтобы создать вариации на основе ваших референсов.

> **Автоматический подбор персонажей по имени файла (опционально):** назовите референсы `dog.png`, `cat.png`, `duck.png`. Промпт, в котором упоминается *«собака играет с кошкой»*, автоматически подтянет `dog.png` и `cat.png`. Идеально для сохранения единообразия персонажей в серии.

### 5. Управление очередью

- Просматривайте все ожидающие задачи в **Очереди промптов**.
- **Удаляйте** ненужные задачи или **Останавливайте** запущенный пакет.
- Отслеживайте прогресс по каждому промпту в реальном времени.

## ⚙️ Справочник по настройкам

Откройте вкладку **Настройки**, чтобы настроить поведение.

**Общие**
- **Режим по умолчанию** — выбирается автоматически при каждом открытии панели.
- **Соотношение сторон по умолчанию** — 16:9, 9:16 или 1:1.
- **Результатов на промпт** — 1–4 видео/изображения на промпт.
- **Одновременных промптов** — выполняйте 1–6 промптов одновременно.
- **Задержка между промптами** — пауза 0–300 с между промптами, чтобы снизить нагрузку на сервер.

**Модели**
- **Видеомодель** — выберите вашу модель VEO (например, VEO 3 / VEO 3 Fast).
- **Модель изображений** — Nano Banana для режимов «текст в изображение» и «изображение в изображение».

**Загрузка**
- **Качество видео** — 720p, 1080p или выключено.
- **Качество изображения** — 1K, 2K, 4K или выключено.
- Файлы сохраняются в папку загрузок Chrome, по одной подпапке на проект.

**Дополнительно**
- **Макс. число повторов** — 1–20 попыток при сбое.
- **Кадр видео по умолчанию** — 8 с или Concat.
- **Макс. изображений на промпт** — задаётся для каждого режима.
- **Язык** — переключение интерфейса между 7 языками.

## 💡 Советы и лучшие практики

**Написание промптов**
- Будьте конкретны в отношении объекта, стиля, движения камеры и освещения.
- Промпты на английском обычно работают наиболее стабильно в Google Flow.
- Разделяйте промпты одной пустой строкой.

**Пропускная способность против стабильности**
- В непиковое время хорошо работает ~3 одновременных промпта.
- В часы пик уменьшите до 2 и повысьте задержку до ~30 с — меньше ошибок «ошибка создания».
- Больше одновременных задач = быстрее, но менее стабильно; подбирайте под себя.

**Управление материалами**
- Называйте референсные изображения понятно, чтобы автоподбор работал.
- Используйте PNG / JPG / GIF, желательно до 10 МБ каждый.

**Надёжность**
- Оставляйте **Автоматическую повторную попытку** включённой для ночных пакетов без присмотра.
- Больше задержка = меньше нагрузка на сервер = меньше сбоев.

## 🔧 Решение проблем

| Симптом | Причина и решение |
| --- | --- |
| **Расширение ничего не делает** | Вы должны находиться на странице **проекта** Google Flow. Убедитесь, что расширение включено, обновите страницу и снова откройте панель. |
| **«Не удаётся создать видео»** | Google Flow временно перегружен. Не волнуйтесь — расширение само ждёт и повторяет попытку каждые ~30 с, пока не освободится слот. |
| **Видео/изображения не скачиваются** | В Chrome: **Настройки → Загрузки → выключите** *«Всегда указывать место для скачивания»*. Затем перепроверьте качество загрузки в Настройках. |
| **«Ошибка политики»** | Ваш промпт или изображение нарушили правила контента Google. Инструмент пропускает эту задачу и автоматически переходит к следующей. |
| **Генерация постоянно сбоит** | Проверьте соединение, убедитесь, что промпты корректны, увеличьте число повторов и просмотрите консоль браузера на предмет подсказок. |
| **Страница сама уменьшила масштаб** | Так задумано — инструмент подстраивает масштаб, чтобы точно находить кнопки. Не меняйте масштаб вручную во время выполнения пакета. |
| **Баннер «отлаживается» не исчезает** | Это нормально во время работы (см. Быстрый старт). Он исчезнет, когда пакет завершится или вы его остановите. |

## ❓ Частые вопросы

**Связан ли VEO Automation с Google?**
Нет. Это независимый сторонний инструмент, который автоматизирует публичное веб-приложение Google Flow. Он не связан с Google, не одобрен и не аффилирован с ним.

**Что такое Nano Banana?**
Nano Banana — это модель изображений, доступная через Google Flow. VEO Automation позволяет пакетно генерировать и автоматически загружать изображения Nano Banana вместе с вашими видео VEO 3 — из одной очереди.

**Могу ли я действительно сгенерировать сотни видео за раз?**
Да. Загрузите файл `.txt` с промптами (или вставьте длинный список), и очередь обработает их один за другим с автоматическими повторными попытками. Реальная пропускная способность зависит от нагрузки Google Flow и ваших настроек одновременного выполнения и задержки.

**Подходит ли это для вертикальных коротких видео?**
Да — установите соотношение сторон **9:16** для TikTok / Reels / Shorts, **16:9** для YouTube или **1:1** для квадрата.

**Куда сохраняются мои файлы?**
В папку загрузок Chrome, разложенными по одной подпапке на проект, с автоматическим переименованием, чтобы их было удобно сортировать.

**Нужен ли мне ключ API?**
Нет. Расширение управляет вашей собственной авторизованной сессией Google Flow в браузере — отдельный ключ API или аккаунт разработчика не требуются.

**Передаются ли мои данные куда-либо?**
Ваши промпты, изображения и видео обрабатываются в вашем браузере на Google Flow. Расширение хранит настройки локально. См. [Конфиденциальность и разрешения](#-конфиденциальность-и-разрешения).

**Какие браузеры поддерживаются?**
Chrome 137+ и браузеры на базе Chromium (Edge, Brave). Требуется поддержка боковой панели.

## 🔒 Конфиденциальность и разрешения

- **Работает в вашем браузере.** Автоматизация происходит в вашей собственной сессии Google Flow.
- **Минимум данных.** Настройки хранятся в локальном хранилище Chrome. Расширение не продаёт и не собирает ваш творческий контент.
- **Ограниченные разрешения.** Доступ к хостам ограничен `labs.google/*` (для автоматизации Flow) и собственным бэкендом расширения (для входа и квоты).
- **Разрешение отладчика** используется исключительно для отправки доверенных событий ввода в Google Flow, которые требуются странице — см. [Быстрый старт](#-быстрый-старт).
